## What is GhostPrint?

GhostPrint is an advanced OSINT (Open Source Intelligence) CLI tool designed for security researchers, penetration testers, and digital forensics investigators.

## Key Features

- **Multi-source Intelligence**: 50+ data sources
- **Fast**: Async operations for concurrent queries
- **Extensible**: Plugin system for custom modules
- **Professional**: Beautiful CLI with Rich
- **Export**: JSON, CSV, HTML reports
